# 基于知识图谱的职业路径推荐系统

## 项目简介
这是一个基于知识图谱的职业路径推荐系统，能够根据用户当前掌握的技能与目标岗位的技能需求，为用户生成个性化的职业发展路径和学习建议。系统通过分析岗位所需技能与用户现有技能的匹配度，提供分阶段的学习计划，帮助用户高效规划职业提升路径。

## 核心功能
- 岗位技能匹配度分析：计算用户技能与各岗位需求的匹配程度
- 个性化学习路径生成：针对目标岗位生成分阶段的技能学习计划
- 可视化技能图谱：直观展示已掌握技能、待学技能及学习阶段关系
- 岗位推荐：基于技能匹配度推荐最适合的岗位（TOP5）
- 简历技能解析：从简历文本中提取技能并写入知识图谱
- 职位图谱浏览：支持岗位关键词搜索及技能关系图谱展示

## 技术栈
- 后端框架：Flask
- 数据库：Neo4j（知识图谱数据库）
- 技能匹配算法：基于技能权重的匹配度计算
- 数据可视化：技能路径关系图谱展示
- 第三方服务：DeepSeek API（用于生成智能分析报告）

## 实现原理
1. 数据存储：使用Neo4j存储岗位（Job）、技能（Skill）和用户（Person）实体，以及它们之间的关系（如岗位需求技能、用户拥有技能）
2. 匹配度计算：通过对比用户技能与岗位所需技能的重合度（结合技能权重）计算匹配率
3. 路径生成：
  - 识别用户已掌握的目标岗位技能和缺失技能
  - 按技能权重排序缺失技能
  - 将缺失技能分为3个学习阶段
  - 针对不同岗位类型（开发/分析/研究等）提供定制化学习建议
4. 可视化展示：构建技能关系图谱，直观展示学习路径

## 接口说明
### 技能联想接口
- 路径：`/api/skill/suggest`
- 方法：GET
- 参数：`prefix` - 技能前缀（用于联想提示）
- 返回：匹配的技能列表（最多10项）

### 职业路径推荐接口
- 路径：`/path-reco`
- 方法：GET（获取页面）、POST（提交目标岗位ID）
- 参数：`job_id` - 目标岗位ID（GET方式从URL获取，POST方式从表单获取）
- 返回：包含岗位推荐列表、技能路径图谱及学习建议的页面

## 使用方法
1. 确保Neo4j数据库已启动并包含必要的岗位和技能数据
2. 配置数据库连接信息（`app.py`中的`NEO4J_URI`、`NEO4J_USER`、`NEO4J_PASSWORD`）
3. 配置DeepSeek API密钥（`app.py`中的`DEEPSEEK_API_KEY`）
4. 安装依赖：`pip install -r requirements.txt`
5. 启动Flask应用：`python app.py`
6. 访问首页，可通过以下功能入口使用系统：
  - 职位图谱浏览：查看岗位及对应技能需求
  - 简历技能解析：上传简历自动提取技能
  - 岗位匹配诊断：输入技能后匹配目标岗位并获取分析报告

## 数据结构
系统在Neo4j中主要使用以下实体和关系：
- 实体：   
  - `Person`：用户，包含属性`id`
  - `Job`：岗位，包含属性`job_id`、`name`、`city`
  - `Skill`：技能，包含属性`name`
- 关系：  
  - `Person-[HAS_SKILL]->Skill`：用户拥有的技能
  - `Job-[REQUIRES]->Skill`：岗位所需的技能（包含`weight`属性表示技能重要性）
